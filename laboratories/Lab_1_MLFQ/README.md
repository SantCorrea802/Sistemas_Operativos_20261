# MLFQ

En este laboratorio se hace una simulación en C del algortimo MLFQ con RoundRobin en cada cola.

## Estructura

Se siguió una estructura similar al entregable anterior, es decir, el reto 2 sobre el Ticket. Con directorios definidos como utils, queue y process con sus respectivos .h y .c, un Makefile que facilita la compilación, ejecución y eliminación del caché y ejecutable.

## Makefile y ejecución

Dentro de el Makefile se encuentran 3 etiquetas:

* all: Encargada de compilar todo el proyecto
* run: Esta ejecuta el ejecutable para lanzar el programa
* clean: Elimina el ejecutable

Para ejecutar alguna de las 3 etiquetas, escriba en la terminal:

make all

make run

make clean
